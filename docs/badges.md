**Requirement**: Physical activity tracking via mobile app

### 3. Developer Badge üíª

**Requirement**: Technical contribution to OpenMind

**Option A: Run a Node** (Easier)
| Step | Action |
|------|--------|
| 1 | Follow the [Node Setup Guide](../README.md) |
| 2 | Run the node and consume ~2,500 credits |
| 3 | Badge is automatically awarded |

**Option B: GitHub Contribution** (More Impactful)
| Step | Action |
|------|--------|
| 1 | Visit [github.com/openmind-ai/OM1](https://github.com/openmind-ai/OM1) |
| 2 | Find issues to work on or propose improvements |
| 3 | Submit a Pull Request (PR) |
| 4 | Get your PR merged |

> üí° **Tip**: Successful PRs are the most effective way to secure this badge and demonstrate real contribution.

**Cost Estimate for Node Method:**
| Metric | Value |
|--------|-------|
| Credits needed | ~2,500 |
| Cost per 100 credits | ~$1 USDC |
| **Total estimated cost** | **~$25 USDC** |
